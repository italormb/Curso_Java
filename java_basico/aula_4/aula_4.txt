Em java o programas são feitos em pacotes,
ao iniciar o java teremos que construir o primeiro pacote. ex: 

package primeiro programa; // Já em pacote é normal que ele seja feito todo em minusculo
public class PrimeiroPrograma {  //observa que as letras maiusculas conseguem identificar e separas as palavras de primeiro e programa
	//TUdo que estiver aqui dentro é uma bloco
	public static void main(String [] args){
     	// metodo principal <
    	// dentro da classe primeiro programa <
    	// que está dentro do primeiro programa como pacote;
    System.out.print("Ola, mundo!");	
	}
}

-JAVA É EM CAMELCASE: MinhaClasse
A primeira letras estão em maiuscula, a primeira letra em maiuscula é classe ou interface e o resto em camelcase, é classe.
-Se a primeira for minuscula e o resto camelcase ela pe um atributo, metodo, variavel ou parametro
- se for tudo minuscula é um pacote
- todas em maiscula é uma constante

- Java é uma linguagem sensivel a maiuscula e minuscula,
